# SHT-Analyse Projekt

Dieses Repository enthält die Auswertung einer groß angelegten Umfrage (960 Teilnehmer) zum Thema Smart Home Technologies (SHT). Um den Ergebnissen die bestmögliche Struktur zu geben, wurde die Auswertung in zwei übersichtliche Berichte aufgeteilt:

1. **[Teil 1: DESKRIPTIV.md](DESKRIPTIV.md)**
   Dieser Bericht bietet einen umfassenden Überblick über das Data Cleaning, die Demografie der Befragten sowie die grundlegenden Meinungen und Interessen an Smart-Home-Produkten. Ideal als tiefergehende Dokumentation.
   
2. **[Teil 2: MARKETING.md](MARKETING.md)**
   Das Management-Summary: Hier werden die Erkenntnisse mittels logistischer Regression in konkrete Business-Insights übersetzt. Welche Faktoren treiben das Kaufinteresse? Was wollen die Kunden kaufen? Das Ergebnis sind datengetriebene Buyer Personas.
